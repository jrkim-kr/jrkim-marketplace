# jrkim-marketplace

Personal Claude Code plugin marketplace by [@jrkim-kr](https://github.com/jrkim-kr).

## Plugins

| Plugin | Description |
|--------|-------------|
| [`flush`](./flush) | Flush changes through the full pipeline — error docs, GitHub issue, conventional commit, push/PR |
| [`architect-advisor`](./architect-advisor) | Senior AI architect & PM advisor — engineering workflow with trilingual term glossary |
| [`my-utils`](./my-utils) | Utility slash commands — daily scrum, doc sync, translations, screenshots, Notion logging, AI builder digest |

## Install

```
/plugin marketplace add jrkim-kr/jrkim-marketplace
/plugin install <plugin-name>@jrkim-marketplace
```

Or, locally:

```
/plugin marketplace add /path/to/jrkim-marketplace
```
